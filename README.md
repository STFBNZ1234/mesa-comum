# Mesa Comum - Gestão de Logística de Alimentos

Este projeto foi desenvolvido para a ONG Mesa Comum com o objetivo de centralizar o controle de entrada de doações, gestão de validade de itens e automação da lista de distribuição, combatendo o desperdício de alimentos.

## Estrutura do Repositório

- `/docs`: Documentação do projeto e Documento de Visão.
- `/backend`: Lógica de servidor e API.
- `/frontend`: Interface do sistema.
- `/database`: Scripts e modelagem do banco de dados.

## Como rodar o projeto

Este repositório contém a estrutura base do sistema. Para instalaçao:

1. Clone o repositorio: `git clone https://github.com/STFBNZ1234/mesa-comum.git`
2. Configure as variàveis de ambiente utilizando o arquivo `.env.example` como base.
3. Instale as dependencias nos diretorios `/backend` e `/frontend` conforme as tecnologias escolhidas.
